Plan Name
Plan Price ($) This is the Starting price of this plan, enter the value without a comma (,)
Plan Minimum Price ($) This is the minimum amount a user can pay to invest in this plan, enter the value without a comma(,)
Plan Maximum Price ($) This is the maximum amount a user can pay to invest in this plan, enter the value without a comma(,)
Total Expected Return ($) This is the total amount a user can expect to earn from this plan, enter the value without a comma(,)
Gift Bonus ($) Optional Bonus if a user buys this plan.enter the value without a comma(,)
Top up Interval This specifies how often the system should add profit(ROI) to user account.
Top up Amount ($) This is the amount the system will add to users account as profit, based on the percentage and interval
Investment Duration This specifies how long the investment plan will run. Please strictly follow the guide below on how to setup investment duration else it may not work. FIRSTLY, the duration are counted in days, so if you want it to go on for months just type in the number of days equal to a month. SECONDLY, do not add anything after the number, LASTLY, if it's weeks, just write 7, a month 30 or 31 and so on if it's a year you put 365.
